📘 How to Prompt Tulu AI for a Great README

  1. Visit Tulu: Go to https://tulu.ai
  2. Find the Prompt Box: Choose the README generator or use the prompt field directly.
  3. Type Your Prompt: Use clear, detailed instructions. For example: 
    Create a professional README file for a GitHub project called "My Portfolio Website".
It is built with HTML, CSS, and JavaScript.
Include: Title, Description, Features, Installation, Usage, Screenshot, Technologies Used, and Contact Info.
  4. Let AI Generate: Submit and wait for the generated output.
  5. Copy to GitHub: Copy the result into a file named README.md and push it to your repository.

Example Prompt:

Generate a detailed README.md for a JavaScript To-Do List web app. Include: Title, Description, Features, Technologies Used, Installation Steps, Usage Guide, Screenshot, and Author Contact.
